YAThemeFiles – YATheme file cross reference

YAThemeFiles objects are used to maintain references between YATheme head files – that is, the file which is at the top of a YATheme rendering tree – and all included files.

It is primarily used for maintaining the YATheme cache.

This object is not instantiated directly. It is designed to be used in the implementatino of YATheme. Other than being an AnInstance object, it is not documented.
